## linux下面操作

```shell
[redis * bin] /usr/alice/databases/redis/redis-3.0.7/bin/redis-server /usr/alice/databases/redis/redis-3.0.7/redis.conf   #启动Redis  
[redis * bin] /usr/alice/databases/redis/redis-3.0.7/bin/redis-cli shutdown save  #关闭Redis 
[redis * bin] ./redis-cli  #登录redis客户端  
[redis * *] keys * get  #查询keys：value值
```
